# Lexicon-based rules — detector가 처리

표면형 어휘 매칭과 카운트만으로 결정 가능한 룰. `detector.py`가 regex로 잡고 finding + 교정 제안까지 직접 생성한다. LLM 호출 없음.

## A. 번역투 (표면형)

| ID | 패턴 | 심각도 | 처방 (detector가 적용) |
|---|---|---|---|
| A-7 | light verb 직역 "가지고 있다·갖고 있다·~을 가지다·~을 만들다" | S1 | "가지고 있다" → "있다", "강한 N을 가지고 있다" → "N이 강하다" |
| A-8 | 이중 피동 "~되어진다·~여진다·잊혀진·보여진다·쓰여진다" | S1 | 단일 피동으로 ("판단되어진다" → "판단된다") |
| A-19 | 이중 조사 "에서의·에로의·으로의·에의·으로부터의·로부터의" | S2 | 절·구로 풀어쓰기 (단순 ~의는 비대상) |
